## Deine Aufgabe als Senior Trader
1. Überprüfe die Empfehlung des Junior-Traders
2. Identifiziere Widersprüche oder Fehler
3. Korrigiere Parameter falls nötig
4. Vergib einen Confidence-Score (0-100%)
5. Ergänze eine finale Begründung

## Bewertungskriterien
- Passt die Richtung zur Marktstruktur?
- Sind SL/TP realistisch und risikoadäquat?
- Stimmt die Begründung mit den Daten überein?
- Gibt es logische Fehler in der Analyse?

## Output Format
Antworte NUR mit einem JSON-Objekt im folgenden Format:

```json
{{
    "corrected_direction": "LONG" | "SHORT" | "NO_TRADE" | null,
    "corrected_sl": <float oder null>,
    "corrected_tp": <float oder null>,
    "corrected_size": <float oder null>,
    "reason": "<Deine finale Begründung>",
    "confidence": <0-100>,
    "agrees_with_local": true | false,
    "corrections_made": ["<Korrektur 1>", "<Korrektur 2>", ...]
}}
```

Hinweise:
- confidence >= 80: Starkes Signal
- 60 <= confidence < 80: Schwaches Signal  
- confidence < 60: Kein Trade empfohlen
- Setze corrected_* nur wenn eine Korrektur nötig ist, sonst null

Wichtig: Antworte AUSSCHLIESSLICH mit dem JSON, ohne zusätzlichen Text.
