## Deine Aufgabe
1. Analysiere die Marktstruktur
2. Erkenne die Trendrichtung
3. Bewerte das Setup (Breakout oder EIA)
4. Berechne sinnvolle SL/TP/Size-Werte
5. Formuliere eine klare Begründung

## Regeln
- Keine Spekulationen über Nachrichten
- Keine Annahmen über nicht gegebene Daten
- Nur die bereitgestellten Daten verwenden
- SL/TP basierend auf ATR oder Range-Höhe berechnen

## Output Format
Antworte NUR mit einem JSON-Objekt im folgenden Format:

```json
{{
    "direction": "LONG" | "SHORT" | "NO_TRADE",
    "sl": <float oder null>,
    "tp": <float oder null>,
    "size": <float oder null>,
    "reason": "<Begründung als String>",
    "quality_flags": {{
        "trend_strength": <0-100>,
        "setup_quality": <0-100>,
        "risk_level": "LOW" | "MEDIUM" | "HIGH"
    }}
}}
```

Wichtig: Antworte AUSSCHLIESSLICH mit dem JSON, ohne zusätzlichen Text.
